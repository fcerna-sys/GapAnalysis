# Ícono (Átomo)

- Nombre: `img2html/atom-icon`
- Clase base BEM: `img2html-icon`
- Propósito: ícono vectorial enmascarado con `currentColor`

## Atributos
- `name`: nombre del ícono (ej. `star`)
- `size`: tamaño en px

## Estructura
```html
<span class="img2html-icon img2html-icon__star" style="width:20px;height:20px"></span>
```
