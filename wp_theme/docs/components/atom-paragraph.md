# Párrafo (Átomo)

- Nombre: `img2html/atom-paragraph`
- Clase base BEM: `img2html-paragraph`
- Propósito: párrafo reutilizable con alineación

## Atributos
- `text`: contenido
- `align`: `left` | `center` | `right`

## Estructura
```html
<p class="img2html-paragraph img2html-paragraph--center">Texto</p>
```
